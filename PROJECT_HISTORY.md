GPTGTA Project History

Overview

GPTGTA is a playable top-down GTA-style browser sandbox developed through iterative AI-assisted development.

The project combines a custom map editor, custom tilesets, vehicles, pedestrians, and browser-based gameplay into a publicly accessible prototype.

Website:

https://gptgta.com

Repository:

https://github.com/vw5hwbngy4-debug/gptgta
